"There's three ways to do things around here, the right way, the wrong way, and the way that I do it."

-Robert Deniro (Casino)

Edit as of 2026:
This section originally contained a very teenage declaration about independence, confidence, and doing things my own way.

It probably said, in so many words, that I did not like being told how to think, act, dress, work, or hold a pencil.

Some of it was likely stubborn. Some of it was probably defensive. But underneath all that, the point was simple: I wanted to be myself, and I wanted that to count for something.

About This Page

The layout and required page titles for this portfolio were set by my high school, so please know I did not personally choose the names of these sections, they are quite clunky if you ask me.

For this page, I picked a quote from Casino that felt like it represented individuality and refusing to bend just because other people expect you to.

There used to be a lot more written here, but it was very clearly written by a teenager, so I removed it before putting this archive back online.
